Bring It!

The following example parses a web-page for images and returns absolute image URLs as an array.

The array is then processed to display the images in HTML.

The alt-text (with the keywords of the content), used by search engines for image-based searches, can be included in the data set.

Assets can be fetched and served locally or assets can be hot-linked like in this example.

The data set can be further processed to only extract data of interest.
